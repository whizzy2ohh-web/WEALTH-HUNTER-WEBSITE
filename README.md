# Wealth Hunter | Hunters Academy

Official website for Hunters Academy — Wealth Hunter.

## Files
- `index.html` — Main website (all pages)
- `styles.css` — All styling and animations
- `script.js` — Interactivity (cursor, nav, FAQ, animations)

## Deploying on GitHub Pages
1. Upload all 3 files to your repository root
2. Go to Settings → Pages
3. Set source to `main` branch, `/ (root)` folder
4. Save — your site will be live at `https://whizzy2ohh-web.github.io`
